# Matrix AI - وثيقة الفكرة 💡

## 🎯 المشكلة اللي نحلو فيها

90% من صناع المحتوى على TikTok و YouTube Shorts يضيعو 4-6 ساعات باش يعملو فيديو 30 ثانية:
1. يكتبو السكريبت
2. يلوجو على صور/فيديوهات
3. يسجلو الصوت
4. يركبو المونتاج
5. يزيدو الـ Subtitles
6. يصدرو الفيديو

**النتيجة**: الإرهاق، وقلة الإنتاجية، وضياع الترند.

## 🚀 الحل: Matrix AI

تطبيق موبايل يختصر الـ 6 خطوات هاذم في **زر واحد**.

**المعادلة:** `جملة واحدة + 60 ثانية = فيديو جاهز للنشر`

### كيفاش يخدم؟
1. **المستخدم يكتب الفكرة**: "قصة نجاح شاب تونسي بدا من الصفر"
2. **يختار اللغة**: 🇹🇳 عربي ولا 🇨🇿 تشيكي
3. **يختار الستايل**: دراما، تحفيزي، مضحك، رعب
4. **يختار الصوت**: راجل ولا مرا
5. **النظام يولد أوتوماتيكيا**:
      - نص القصة بـ GPT-4o
      - 4-6 صور بـ Stable Diffusion XL
      - تعليق صوتي بـ ElevenLabs
      - موسيقى خلفية
      - ترجمة Subtitles مدمجة
      - مونتاج كامل بـ FFmpeg
6. **النتيجة**: فيديو MP4 بجودة 1080x1920 جاهز للتنزيل

## 🌍 ميزة اللغتين: 🇹🇳 + 🇨🇿

### 1. في الواجهة
- المستخدم يبدل بين `عربي | Čeština` بضغطة وحدة تحت اللوغو
- **العربي**: RTL كامل، خط Cairo، أصوات تونسية/مصرية/خليجية
- **التشيكي**: LTR كامل، أصوات تشيكية طبيعية مثل `Antonín` و `Zuzana`

### 2. في المحتوى
- **Prompt Engineering منفصل**: الـ AI يكتب قصص تناسب الثقافة التونسية ولا التشيكية
- **الترند**: قسم "ترند اليوم" يجيب الترندات من تيك توك تونس ولا تيك توك التشيك حسب اللغة

## ✨ المميزات الأساسية لـ MVP

| الميزة | الوصف |
| --- | --- |
| **زر واحد = فيديو** | توليد كامل من الفكرة للفيديو النهائي |
| **أصوات طبيعية** | ElevenLabs بلهجات متعددة |
| **Subtitles أوتوماتيكية** | مدمجة ومزامنة مع الصوت |
| **قوالب ترند** | فيديوهات جاهزة حسب المواضيع الطالعة |
| **تصدير HD** | 1080x1920 مناسب لـ TikTok/Shorts/Reels |
| **دعم RTL/LTR** | العربية من اليمين لليسار، التشيكية من اليسار لليمين |

## 🛠️ التقنيات المقترحة

- **Frontend**: Flutter - باش نطلعو iOS + Android بكود واحد
- **Backend**: Python + FastAPI - سريع وقوي للـ AI
- **قاعدة البيانات**: Supabase - فيها Auth + Storage + Database
- **AI Stack**:
    - النص: GPT-4o
    - الصور: Stable Diffusion XL
    - الصوت: ElevenLabs
    - المونتاج: FFmpeg
